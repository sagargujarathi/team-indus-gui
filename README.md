## REQUIREMENTS TO RUN THE PROJECT

### FOLLOW THE COMMANDS

1. DOWNLOAD NVM USING (THIS LINK)[https://github.com/coreybutler/nvm-windows/releases/download/1.2.2/nvm-setup.exe].
2. OPEN CMD.
3. `nvm install lts`
4. `nvm use lts`
5. `npm install`
6. `npm run dev`
